## Column headers

Where present, the first row of every CSV is a header line documenting each
column. DAT files follow the `pgfplots` convention with `# col1 col2 col3 …`
on the first commented line. LOG files are free-form text records of the run
and are kept for traceability of mesh/ensemble convergence.

## Regenerating

To regenerate every output, from the `code/` root:

```bash
for s in scripts/0[0-9]_*.py; do
  python -m scripts.$(basename "$s" .py)
done
```

Final outputs are reproducible up to floating-point determinism issues; the
relative tolerances reported in the article are achieved at the resolutions
documented in the head of each script.
